## Flexbox card wrapper
```css
<div class="flex-grid-wrapper">
    <div class="flex-grid">
        <!-- cards here -->
        <div class="flex-grid-item">
            item 1
        </div>
        <div class="flex-grid-item">
            item 2
        </div>
        <div class="flex-grid-item">
            item 3
        </div>
    </div>
</div>
```

#### Settings
```css
--grid-row-items-n: 3; // number of cards on a row
--grid-row-items-p: 20px; // margin between cards
```
